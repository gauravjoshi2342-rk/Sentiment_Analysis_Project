Social Media Sentiment Analysis and Visualization

Hi there! You've stumbled upon my project, an end-to-end Exploratory Data Analysis (EDA) and Sentiment Analysis project hosted in this repository. Being a Computer Science student with an avid interest in AI and Machine Learning, I created this tool to delve into the emotional spectrum and interaction habits of people on various social media platforms.

The Project

In this endeavor, I sought to transform the raw, noisy data generated on social media into meaningful, decipherable patterns. The project involved an exhaustive cleaning process of the dataset, followed by a thorough sentiment categorization of the user generated text, and the ultimate visualization of how various demographics interacted with this content to identify engaging trends.

Tech Stack

*   Language: Python 3.x

*   Data Processing: Pandas, NumPy

*   Visualization: Matplotlib, Seaborn

*   NLP/Sentiment: Custom classification logic

*   Environment: Jupyter Notebook ( VS Code )

Key Highlights

*   Deep EDA: Explored correlation matrices and various distribution patterns in the data to comprehend the engagement behaviors of various demographics.
*   Sentiment Engineering: Categorized emotions into three classes ( Positive, Negative and Neutral ) for easier analysis of complex human generated text.

*   Advanced Visuals:

*   Platform Comparison: Utilized Grouped Bar Charts to examine and contrast sentiments across different platforms.
    *   Engagement Metrics: Leveraged Scatter plots and Histograms to evaluate the correlation between 'Likes' and 'Retweets'.
    *   Trend Tracking: Analyzed monthly content performance through Line Charts to discern engagement fluctuations.
    *   Result Export: Configured professional image export formats for clean documentation and reporting.
